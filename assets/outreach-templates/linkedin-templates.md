# LinkedIn Outreach Templates

## Connection Request (under 300 chars)

### Template 1: Direct
```
Hi [Name], I help [niche] automate [task] to save 15+ hours/week. Worth connecting?
```

### Template 2: Value-First
```
Hi [Name], I noticed [specific thing about their business]. I share insights on AI automation for [niche]. Let's connect.
```

### Template 3: Mutual Interest
```
Hi [Name], saw your post on [topic] — great perspective. I work on AI automation for [niche]. Would love to connect.
```

## Follow-Up Message (after connection accepted)

### Template 1: Value-Add
```
Hi [Name], thanks for connecting!

I've been following your work on [topic] — really insightful.

Quick question: How much time does your team spend on [task] weekly?

I help [niche] automate this, typically saving 15-20 hours/week. Happy to share ideas if relevant.
```

### Template 2: Case Study
```
Hi [Name], thanks for connecting!

I recently helped [similar company] automate [task] — they went from [problem] to [result] in 2 weeks.

Would love to hear how you handle [task] at [their company].
```

## Engagement Comments (on their posts)

### Template 1: Add Value
```
Great point about [topic]. I've seen this work well when combined with [automation tactic]. The results can be significant.
```

### Template 2: Ask Question
```
Interesting perspective. Have you considered how automation could amplify these results? I've seen 3x improvements in similar setups.
```

### Template 3: Share Experience
```
This resonates with what I've seen working with [niche]. The businesses that adopt this early tend to see the biggest gains.
```
